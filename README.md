## Step1

### 요구사항
* 스토리보드 ViewController에 CollectionView를 추가하고 Safe 영역에 가득 채우도록 frame을 설정한다.

* CollectionView Cell 크기를 80 x 80 로 지정한다.

* UICollectionViewDataSource 프로토콜을 채택하고 40개 cell을 랜덤한 색상으로 채우도록 구현한다.
